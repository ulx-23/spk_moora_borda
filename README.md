# SPK MOORA BORDA

## 📌 Description
SPK MOORA BORDA is a web-based Decision Support System developed using CodeIgniter 3 and MySQL. This system implements the MOORA (Multi-Objective Optimization on the Basis of Ratio Analysis) and BORDA methods to support objective, accurate, and efficient decision-making processes based on multiple criteria and alternative rankings.

---

## 🚀 Features
- Authentication Login System
- Dashboard Admin
- Criteria Management
- Alternative Management
- Weight Management
- MOORA Calculation
- BORDA Ranking
- Final Ranking Results
- CRUD Functionality
- Responsive Interface

---

## 🛠️ Technologies Used
- PHP
- CodeIgniter 3
- MySQL
- Bootstrap
- HTML5 & CSS3
- JavaScript
- jQuery

---

## 📊 Methods Used

### 🔹 MOORA Method
MOORA is used to calculate alternative preference values through normalization and optimization processes.

### 🔹 BORDA Method
BORDA is used to combine ranking results and determine the final recommendation objectively.
